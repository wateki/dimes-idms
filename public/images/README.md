## Image Guidelines

1. **Format**: PNG or JPG (PNG preferred for screenshots with transparency)
2. **Optimization**: Compress images for web (use tools like TinyPNG or ImageOptim)
3. **Style**: Professional, clean, enterprise-grade appearance
4. **Colors**: Should complement the emerald green (#10B981, #059669) color scheme
5. **Content**: Show real platform features, dashboards, and interfaces

## Placeholder Behavior

The code includes fallback placeholders that will display if images are not found. Once you add the actual images, they will automatically replace the placeholders.
